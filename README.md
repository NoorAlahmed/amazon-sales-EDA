# Amazon Sales Dataset - EDA Project

## Dataset Source
This dataset was obtained from Kaggle and contains information about Amazon products including pricing, ratings, and categories.

## Number of Rows and Columns
The dataset contains  1465 rows and 16 columns.

## Features Description
product_id - Product ID
product_name - Name of the Product
category - Category of the Product
discounted_price - Discounted Price of the Product
actual_price - Actual Price of the Product
discount_percentage - Percentage of Discount for the Product
rating - Rating of the Product
rating_count - Number of people who voted for the Amazon rating
about_product - Description about the Product
user_id - ID of the user who wrote review for the Product
user_name - Name of the user who wrote review for the Product
review_id - ID of the user review
review_title - Short review
review_content - Long review
img_link - Image Link of the Product
product_link - Official Website Link of the Product

## Purpose of the Dataset
The purpose of this dataset is to perform exploratory data analysis (EDA) to understand patterns in product pricing, customer ratings, and sales trends.

## Notes
- Data cleaning was performed to handle missing values and incorrect formats.
- Visualizations were created to extract meaningful insights.
